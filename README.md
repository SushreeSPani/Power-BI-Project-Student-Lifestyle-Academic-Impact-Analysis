# Power-BI-Project-Student-Lifestyle-Academic-Impact-Analysis

 Key Insights from the Dashboard:
 • 705 students analyzed
 • Average usage: 4.92 hours/day
 • 64.26% reported academic impact
 • Average sleep: 6.87 hours
📈 The dashboard explores:
Usage trends across age groups
Addiction score by academic level
Gender-based distribution insights
Correlation between screen time, sleep, and academic performance
